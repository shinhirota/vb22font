# VOLLEY BALL 2on2 (vb22font)

## 概要
往年のPC-9801用フリーソフト『VOLLEY BALL 2on2』に使われていたフォントデータを抽出し、Cocos2d-x などで使用可能な fnt ファイルにしたのでここに置いておきます。
16x16のアルファベット等 (vb22font.fnt) と、16x32のスコア表示用の数字 (tallnumbers.fnt) があります。
（どう考えても需要はありませんが・・・）
